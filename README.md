# Future-maker
학교생활을 하며 미래를 결정하는 시뮬레이션 게임

### 개발자
팀 명 : 
|이름|전공|
|----|----|
|신애리|빅데이터 21|
|김수연|빅데이터 21|

### 구현 분담
|이름|구현 내용|
|----|----|
|신애리|체육관, 도덕실, 학교, 아르바이트 장소|
|김수연|집, 음악실, 영어실, 공원, 식당|

### 프로젝트 개요

- 유니티를 이용하여 학업, 아르바이트 등의 상호작용과 각각의 미니게임을 하며 성적을 향상시키고 1년간의 과정을 거쳐 졸업을 하는 생활 시뮬레이션 게임이다.
